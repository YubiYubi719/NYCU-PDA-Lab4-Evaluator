## Preparation
1. Put all input files and your `.lg` file into a directory
2. Make sure that all of your files have the same names

    For example:
    ```
    📁publicCase/  
        └── testcase0.gmp  
        └── testcase0.gcl
        └── testcase0.cst
        └── testcase0.lg
    ```

## Execution
```
$ tar xvf Evaluator.tar
$ cd Evaluator
$ chmod 755 Evaluator
$ ./Evaluator <fileDirPath> <testcaseName>

// Ex:
// ./Evaluator ~/publicCase testcase0
```
